## Run Locally

Clone the project

```bash
  git clone https://github.com/jouryfirst/organized-box.git
```

Go to the project directory

```bash
  cd organized-box
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```
